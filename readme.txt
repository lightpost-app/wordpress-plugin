This plugin allows churches to display content from their Lightpost account on their Wordpress-based website.

== Description ==

This plugin allows churches to display specific content from their [Lightpost](https://lightpost.app) account on their Wordpress-based website. For example, it will display public sermons from the sermon archive or Bible classes from the Bible class registration options. This is a flexible, easy way to keep this content up-to-date on your church's website.

Please note: you must be a Lightpost customer to use this plugin. This plugin can be used with any Wordpress theme. To become a Lightpost customer, or to simply learn more about Lightpost, please visit the website at [lightpost.app](https://lightpost.app).

== Installation ==

- Upload the plugin into a `lightpost` folder in the `/wp-content/plugins` directory.
- Activate plugin in the Wordpress admin.
- In the Wordpress admin, go to Settings, then Lightpost, and enter your Lightpost API key.
- To enable the diretory, Bible class, calendars or sermon archives, simply select the pages you want them to appear on.
- You can also optionally set a theme.

== Frequently Asked Questions ==

= What is Lightpost? =

[Lightpost](https://lightpost.app) is a software suite built specifically for the Lord's church. It allows church administrators to manage many aspects of their church data, easily and securely. It also provides church members access to specific church data, via the website or mobile application, where they can use helpful features like the member directory, Bible class registration, involvement areas, calendars, account information, sermon archives, group messaging, and more.

= How do I use this plugin? =

This plugin works once you have signed up for a Lightpost account and can access the API key for your account.
